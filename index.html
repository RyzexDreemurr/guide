  • Monitor network activity when running new software
  • Use virtual machines for testing potentially dangerous files

Identifying Malicious Software

Be vigilant for these red flags after running any software:

System Behavior

  • Random CMD windows flashing
  • Browsers closing unexpectedly
  • Sudden PC slowdowns/crashes
  • Unusual network activity

Process Monitoring

  • Unknown processes in Task Manager
  • Suspicious startup programs
  • Antivirus disabled or alerting
  • High CPU/GPU usage when idle

Immediate Action Required

If you notice any of these signs:

  1. Disconnect from the internet immediately
  2. Run a full system scan with Windows Defender
  3. Check startup programs using Autoruns
  4. Consider resetting your PC if infection is confirmed

Diagnostic Tools

  • Process Explorer
  • Autoruns
  • Wireshark

Prevention

  • Use standard user account
  • Keep OS & software updated
  • Use reputable antivirus
  • Backup important files

Common Scams

  1. Unverified Executors (TikTok, DMs)
  2. HAR File Requests (NEVER SHARE)
  3. Phishing Links
  4. "Free Nitro"/Robux Generators
  5. Fake Livestream Giveaways
  6. Fake "Roblox Support"
  7. Suspicious QR Code Scans

General Security Practices

  • NEVER share .ROBLOSECURITY cookie or password.
  • Use strong, unique passwords + 2FA (Authenticator App). Use Password Manager.
  • Enable Account PIN on Roblox.
  • Use Verification Tools & sandboxes.
  • Be skeptical of unrealistic offers.
  • Check authorized sessions regularly.
  • Monitor account activity logs.
  • NEVER run obfuscated scripts from untrusted sources.
  • Be wary of .lua files from DMs/random sites.
  • Check scripts for require(), loadstring, getfenv, webhook, etc.
  • Keep OS & browser updated.
  • Don't disable antivirus or run random commands.
  • Avoid untrusted Roblox "mods" or extensions.

Account Protection

  • Use unique email
  • Never reuse passwords

  • Verify email/phone

Discord Safety
